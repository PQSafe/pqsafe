Privacy Policy

Effective date: 2026-05-02
TL;DR — pqsafe.xyz collects no personal data. No analytics, no cookies, no tracking pixels, no third-party scripts. This is a static site backed by fully open-source code.

1. Data we collect

None. pqsafe.xyz does not collect, store, or transmit any personal information. There are no sign-up forms, no accounts, and no user-submitted data of any kind on this website.

2. Cookies & local storage

This site sets no cookies and uses no browser storage (localStorage, sessionStorage, IndexedDB). There are no consent banners because there is nothing to consent to.

3. Analytics & tracking

We run no analytics platform, no tracking pixels, and no third-party JavaScript analytics (e.g. Google Analytics, Plausible, Mixpanel, Segment, or similar). The page source is fully inspectable and contains no third-party script tags.

4. Email contacts

If you email us at raymond@pqsafe.xyz or tris@pqsafe.xyz, we will use your email address and message solely to reply to your inquiry. We do not add you to mailing lists, share your address with third parties, or retain correspondence beyond what is necessary to resolve your query.

5. GitHub & npm packages

PQSafe's source code is published at github.com/PQSafe/pqsafe and distributed as npm packages under standard open-source metadata (package name, version, author, license). These registries may collect standard request logs (IP address, timestamp) subject to their own privacy policies:

  • GitHub Privacy Statement
  • npm Privacy Policy

We do not embed telemetry in any published package. No PII is included in package metadata.

6. Hosting infrastructure

pqsafe.xyz is a static site hosted on GitHub Pages. GitHub's edge infrastructure may log standard request metadata (IP address, User-Agent, timestamp) at the network level. This is outside our control; see the GitHub Privacy Statement linked above.

7. Third-party services

This site loads no third-party fonts, no CDN-hosted scripts, and no embedded iframes. All assets are served from pqsafe.xyz directly.

8. Children

PQSafe is a general-purpose post-quantum cryptography tool and is not directed at children under 13. We do not knowingly collect information from minors.

9. Changes to this policy

If this policy changes, the updated version will be published here with a new effective date and announced in the project's GitHub repository. All existing commitments remain in force.
