What's monitored

  • Verifier API — the canonical endpoint that validates signed envelopes on the server side.
  • Issuer — the controlled service that signs SpendEnvelopes with the canonical PQSafe keypair.
  • Sandbox — the public test endpoint at sandbox.pqsafe.xyz (test issuer, public test keys, no replay enforcement).
  • On-chain registry — the SpendEnvelope nonce + revoke registry contract on Arbitrum Sepolia.
  • Test vectors — the published AP2-PQ v1 conformance vectors at /spec/ap2-pq-test-vectors-v1.json.
  • Demo — the live end-to-end demo at demo.pqsafe.xyz.
